# __PROJECT_TITLE__

NestJS API generated by `create-onion-nest` with Onion/Clean Architecture defaults.

## Architecture

```text
src/domain          Enterprise business rules: entities and repository contracts
src/application     Application business rules: use cases
src/infrastructure  Framework and external adapters: persistence, clients
src/presentation    NestJS modules and controllers
src/shared          Cross-cutting primitives
```

Dependencies point inward: presentation and infrastructure depend on application/domain contracts, while domain does not import NestJS.

## Commands

```bash
npm install
npm run start:dev
npm run build
npm test
```

## Add a Module

From this project root or from another directory with `--cwd`:

```bash
npx create-onion-nest module billing
npx create-onion-nest module billing --cwd ./__PROJECT_NAME__
```
